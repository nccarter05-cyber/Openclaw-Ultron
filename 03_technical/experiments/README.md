# Experiments

Tests, hypotheses, proof-of-concepts.

One folder per experiment.

Each experiment:
- `hypothesis.md` - What you're testing and why
- `setup.md` - How you set it up
- `results.md` - What happened, what you learned

After an experiment:
- Move to lessons if it's a pattern to remember
- Move to agents/ or integrations/ if it becomes a product
- Move to 05_archive/ if it's done and won't repeat

---

*Build fast, learn faster.*
